# Interactive Personal Portfolio Website

A responsive personal portfolio website enhanced with JavaScript to provide smooth navigation, animations, and user interaction.

---

## Project Description

This project is an interactive personal portfolio designed to showcase skills, education, achievements, and contact information.  
JavaScript is used to enhance user experience through smooth scrolling, section animations, dynamic skill indicators, theme switching, and form feedback.

The focus of this project is usability, accessibility, and clean interaction without using external frameworks.

---

## Features

- Smooth scrolling navigation
- Active navigation highlighting based on scroll position
- Section reveal animations on scroll
- Dynamic skill level indicators
- Contact form validation with user feedback
- Back-to-top button
- Light and dark theme toggle
- Responsive layout for all devices

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## How It Works

- Navigation links smoothly scroll to sections
- The active section is highlighted in the menu
- Sections fade in as they enter the viewport
- Skill levels are dynamically generated using JavaScript
- Contact form validates input and shows status messages
- Theme preference is saved using local storage
- Back-to-top button appears after scrolling
